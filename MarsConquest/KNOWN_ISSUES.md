# Mars LogBook - известные технические вопросы

Последнее обновление: 2026-07-14

Этот файл содержит только подтверждённые технические риски. Планируемые функции и улучшения интерфейса находятся в `TASKS.md`.

## Активные вопросы

### Предупреждение каталога ассетов

Xcode сообщает, что в `background_image.imageset` есть 15 нераспределённых изображений. Приложение собирается и работает, но при следующей чистке ресурсов нужно либо назначить этим изображениям роли в `Contents.json`, либо перенести/удалить ненужные файлы.

### Обновления Core Data

При каждом будущем изменении схемы Core Data нужно создавать новую версию модели и проверять обновление на устройстве с уже сохранёнными партиями. Текущая миграция с `GameDataModel` на `GameDataModel 2` проверена сборкой и запуском на устройстве.

## Закрытые исторические проблемы

Награды и достижения сохраняются, участвуют в итоговом счёте и отображаются в деталях игры. Повторный запуск загрузчика справочников не создаёт дубликаты. Устаревшие `presentationMode` и навигационный API удалены.

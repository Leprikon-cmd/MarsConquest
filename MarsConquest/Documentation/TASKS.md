# Mars LogBook - активные задачи

Последнее обновление: 2026-07-17

Этот файл содержит только незавершённые задачи. История уже сделанной работы собрана в `WORK_HISTORY.md`.

## Направление продукта

Mars LogBook прежде всего остаётся личным локальным журналом партий. История и статистика принадлежат игроку и работают без сервера.

Будущие сетевые функции строятся вокруг закрытых групп друзей, а не публичного рейтинга без подтверждения результатов.

## P1 - ближайшие задачи

- [ ] Расширить стабильные идентификаторы игрового справочника на поля, награды и достижения; сохранять их в новых партиях вместе с названиями и оставлять запасной вариант для старой истории.
- [ ] Добавить русские и английские отображаемые имена игрового справочника после завершения работы с идентификаторами.
- [ ] Провести визуальную проверку английского режима на маленьком и крупном iPhone: длина текста, даты, числа, пустые состояния и системные диалоги.
- [ ] После завершения текущего цикла тестов убрать автоматический выбор корпорации и прологов: поля должны быть пустыми, а добавление игрока недоступно до явного выбора.

## P2 - дизайн и удобство

- [ ] Перерисовать кнопки «Достижения» и «Награды» в стиле настольной игры, используя подходящие изображения вместо однотонной заливки.
- [ ] Привести экраны новой партии, подсчёта и деталей игры к единому спокойному стилю: одинаковые отступы, заголовки, кнопки и цвета.
- [ ] Улучшить подачу итогов: сделать победителя и рейтинг заметнее, не перегружая экран.
- [ ] После стабилизации интерфейса проверить его на маленьком и крупном iPhone.

## P4 - компаньон партии и контроль времени

- [ ] Добавить в настройки правил партии переключатель «Драфт» и сохранять этот выбор вместе с игрой.
- [ ] Добавить необязательный режим «Компаньон партии»: первый игрок, порядок хода, передача телефона, «Пасс» и окончание поколения.
- [ ] Добавить мягкий таймер партии: общее время, предупреждения и журнал длительности поколений.
- [ ] Добавить шахматный режим с личным запасом времени каждого игрока.
- [ ] Продумать домашние правила для превышения времени: предупреждение, пометка или строгий режим с подтверждением организатора.
- [ ] Сохранять в статистике время партии, число поколений, включённый драфт и выбранный режим контроля времени.

## P5 - закрытые группы и подтверждение партий

- [ ] Добавить резервную копию и перенос личной локальной истории между устройствами.
- [ ] Дать пользователю создать закрытую игровую группу и пригласить друзей.
- [ ] Реализовать общую партию группы: каждый участник вносит свои данные, а инициатор видит итоговую сводку.
- [ ] Добавить подтверждение результата всеми участниками; партия записывается в локальную историю каждого только после согласия группы.
- [ ] Продумать исправление и повторное подтверждение результата.
- [ ] Хранить режим правил партии: официальные правила или домашние правила / вне рейтинга.

## P6 - клубы и турнирный режим

- [ ] Добавить роль организатора / судьи, который вносит и подтверждает результат турнирной партии.
- [ ] Фиксировать состав игроков, официальный набор дополнений и правила турнира.
- [ ] Вести отдельную подтверждённую статистику турниров и клубов.
- [ ] Добавить журнал подтверждений и исправлений для спорных результатов.

## Очередь работы

1. Завершить стабильные ключи игрового справочника и безопасный перевод игровых названий.
2. Визуальная проверка английского режима.
3. Единый визуальный стиль и игровые кнопки.
4. Компаньон партии и контроль времени.

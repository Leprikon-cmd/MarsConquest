# Mars LogBook - контекст проекта

Последнее обновление: 2026-07-14

## Назначение

Mars LogBook - не самостоятельная цифровая версия настольной игры, а локальный iPhone-компаньон для проведения партии: он собирает результаты, считает итоговые очки и хранит историю игр.

## Что уже работает

- создание партии с выбором поля и дополнений;
- добавление сохранённых и новых игроков с уникальными именами;
- выбор корпораций и прологов;
- подсчёт очков по категориям;
- достижения и награды со всеми текущими правилами ничьих;
- общий рейтинг игроков;
- сохранение в Core Data, импорт партий и просмотр деталей;
- статистика по играм, игрокам, корпорациям и прологам;
- опциональная фоновая дорожка МОКСИ;
- запуск и проверка на физическом iPhone.

## Текущая архитектура

SwiftUI -> `LocalGameData` -> `GameSaver` -> Core Data -> статистика и детали партии.

`GameData.swift` содержит встроенный каталог. `InitialDataLoader` переносит его в Core Data без дубликатов, а интерфейс использует Core Data как рабочий справочник.

## Ближайшие направления

1. P1: удобство ввода очков, расширения и подготовка локализации.
2. P2: редизайн деталей партии, таблицы подсчёта и игровых кнопок.
3. P4: режим компаньона партии с драфтом и контролем времени.

Полная актуальная очередь находится в `TASKS.md`. Техническая схема описана в `DATABASE_SCHEMA.md`, а устройство кода - в `PROJECT_ARCHITECTURE.md`.
